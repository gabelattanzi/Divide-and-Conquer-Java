# Divide-and-Conquer-Java

A Java-based implementation of a divide-and-conquer algorithm based on merge sort logic, which counts the number of inversions done to an array while simultaneously sorting it.

It runs in _O(nlogn)_ time, and will display both the sorted array and number of inversions back to you after execution.

To use it, simply put your unsorted array into a text file within the root directory of the program, formatted like in "sample_input.txt". Then type the file name (including the extension) into the prompt given.
